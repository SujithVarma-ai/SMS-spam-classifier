# 🛡️ SMS Spam Classifier

A professional and deployment-ready Streamlit web application that uses **Machine Learning** and **Natural Language Processing (NLP)** to classify SMS messages as either **Spam** or **Ham (Safe)**.

## 🌐 Live Demo

🔗 https://sms-spam-classifier-xc9nttah6fe2k9xspzzfhu.streamlit.app/

---

## 🚀 Features

- SMS Spam Detection using Machine Learning
- TF-IDF Vectorization
- NLP preprocessing with NLTK
- Interactive Streamlit Web Interface
- Real-time Prediction
- Lightweight and Fast
- Cloud Deployment Ready
